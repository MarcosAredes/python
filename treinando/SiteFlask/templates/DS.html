Data Science

O que e Data Science(Ciência de dados):

Data Science é uma disciplina multidisciplinar que envolve a extração de informações úteis e insights a partir de dados. Ela combina conhecimentos de estatística, matemática, programação, ciência da computação e domínio do assunto para analisar conjuntos de dados complexos.


O processo de Data Science geralmente envolve as seguintes etapas:

1°Coleta de Dados:
Obtenção de dados relevantes para o problema em questão. Os dados podem ser estruturados (como tabelas em bancos de dados) ou não estruturados (como texto, imagens ou áudio).

2°Limpeza de Dados:
Trata-se de lidar com dados ausentes, inconsistentes ou incorretos. Isso inclui a remoção de outliers e a normalização de dados.

3°Exploração de Dados (EDA):
Compreensão inicial dos dados por meio de estatísticas descritivas, visualizações e outras técnicas exploratórias. O EDA ajuda a identificar padrões, tendências e relações nos dados.

4°Preparação de Dados:
Transformação dos dados em um formato adequado para análise. Isso pode envolver a criação de features (variáveis) adicionais, codificação de variáveis categóricas, normalização, etc.

5°Modelagem:
Desenvolvimento de modelos estatísticos ou algoritmos de machine learning para fazer previsões ou tomar decisões com base nos dados.

6°Avaliação do Modelo:
Avaliação do desempenho do modelo em dados não vistos. Isso ajuda a garantir que o modelo seja capaz de generalizar bem para novos dados.

7° Interpretação e Comunicação dos Resultados:
Compreensão dos resultados obtidos e comunicação desses resultados de maneira clara e compreensível para as partes interessadas.

8°Iteração:
Com base nos resultados e nas necessidades do projeto, o cientista de dados pode iterar em várias etapas do processo para melhorar a qualidade dos resultados.

Data Science é amplamente aplicada em diversas áreas, incluindo finanças, saúde, marketing, ciências sociais, entre outras. A análise de grandes conjuntos de dados permite identificar padrões e tendências que podem levar a uma tomada de decisão mais informada e à resolução de problemas complexos.

Dados

Em ciência de dados, o termo "dado" refere-se a informações ou fatos que podem ser coletados, armazenados e analisados. Os dados são a matéria-prima fundamental para os cientistas de dados, pois são a base para a obtenção de insights, tomada de decisões e construção de modelos preditivos. Aqui estão alguns conceitos importantes relacionados a dados em ciência de dados:
Tipos de Dados:

    Dado Estruturado: Dados organizados em tabelas ou formatos tabulares, como bancos de dados relacionais.

    Dado Não Estruturado: Dados que não têm uma estrutura fixa, como texto não formatado, imagens, áudio e vídeo.

    Dado Semiestruturado: Dados que não se encaixam completamente em um formato estruturado, mas têm alguma estrutura, como documentos XML ou JSON.
Fontes de Dados:
    Dado Primário: Dados coletados diretamente de fontes originais por meio de experimentos, pesquisas ou observações.

    Dado Secundário: Dados que foram coletados por outra pessoa ou organização para propósitos diferentes dos seus, mas que podem ser relevantes para a sua análise.
Processamento de Dados:
    Limpeza de Dados: Remoção de valores ausentes, tratamento de outliers e correção de erros nos dados para garantir qualidade.

    Transformação de Dados: Conversão de dados em um formato mais adequado para análise, como normalização ou codificação de variáveis.
Medidas Estatísticas:
    Média: Valor médio de um conjunto de dados.

    Mediana: Valor do meio em um conjunto ordenado de dados.

    Desvio Padrão: Medida de dispersão que indica quão espalhados estão os valores em torno da média
Visualização de Dados:
    Gráficos: Representações visuais dos dados, como histogramas, gráficos de dispersão e box plots.

    Dashboards: Painéis interativos que consolidam informações importantes de forma visual.
Aprendizado de Máquina:
    Treinamento de Modelo: Utilização de dados para ensinar um algoritmo a fazer previsões ou tomar decisões.

    Conjunto de Treinamento e Teste: Divisão dos dados em partes para treinar o modelo e testar sua performance
Privacidade e Ética:
    Privacidade dos Dados: Garantir que os dados sejam tratados de maneira ética e que a privacidade das pessoas seja preservada.

    Bias nos Dados: Viés que pode surgir devido a desigualdades nos dados, levando a resultados injustos.
Em resumo, os dados desempenham um papel crucial em todas as fases do processo de ciência de dados, desde a coleta até a interpretação dos resultados. A qualidade, integridade e relevância dos dados são fundamentais para garantir análises e tomadas de decisão precisas e confiáveis.

Documentação do Pandas https://pandas.pydata.org/docs/

O que é:

Pandas é uma biblioteca de software escrita para a linguagem de programação Python para manipulação e análise de dados. Ela fornece estruturas de dados eficientes e fáceis de usar, como o DataFrame, que é uma tabela bidimensional com rótulos em linhas e colunas. O Pandas é amplamente utilizado em ciência de dados, análise financeira, estatísticas e outras áreas onde a manipulação de dados é uma parte essencial do processo.

Algumas características principais do Pandas incluem:

DataFrame:

    Uma estrutura de dados tabular bidimensional com rótulos em linhas e colunas, semelhante a uma planilha.
Series:
    Uma estrutura de dados unidimensional que pode armazenar qualquer tipo de dado.
Operações de dados:
    O Pandas oferece uma ampla variedade de funções e métodos para realizar operações de manipulação, limpeza e análise de dados.
Leitura e escrita de dados:
    O Pandas suporta a leitura e escrita de dados em vários formatos, como CSV, Excel, SQL, HDF5, e outros.
Indexação sofisticada:
    O Pandas permite indexar dados de maneiras complexas, facilitando a recuperação e manipulação de informações específicas.

O Pandas é uma ferramenta essencial em muitos projetos de ciência de dados e análise de dados em Python devido à sua eficiência, flexibilidade e facilidade de uso.

Se voce digitar pd.read_ tem varias variedade de arquivos para que se possa utilizar para leitura.

Aqui vão os pd.read_:

pd.read_clipboard()

    Descrição: Lê o conteúdo da área de transferência e o converte em um DataFrame.

    Code:
    import pandas as pd
    df = pd.read_clipboard()

pd.read_csv()
    Descrição: Lê um arquivo CSV (Comma Separated Values) em um DataFrame do pandas.

    Code
    import pandas as pd
    df = pd.read_csv('seu_arquivo.csv')
pd.read_excel()
    Descrição: Lê um arquivo Excel em um DataFrame do pandas.

    Code
    import pandas as pd
    df = pd.read_excel('seu_arquivo.xlsx')
pd.read_feather()
    Descrição: Lê um arquivo Feather em um DataFrame do pandas.

    Code
    import pandas as pd
    df = pd.read_feather('seu_arquivo.feather')
pd.read_fwf()
    Descrição: Lê um arquivo de largura fixa (Fixed Width Format) em um DataFrame do pandas

    Code
    import pandas as pd
    df = pd.read_fwf('seu_arquivo.txt')
pd.read_gbq()
    Descrição: Lê um conjunto de dados do Google BigQuery em um DataFrame do pandas.

    Code
    import pandas as pd
    project_id = 'seu-projeto-id'
    sql = 'SELECT * FROM seu_dataset.sua_tabela'
    df = pd.read_gbq(sql, project_id=project_id)
pd.read_hdf()
    Descrição: Lê um arquivo HDF5 em um DataFrame do pandas.

    Code
    import pandas as pd
    df = pd.read_hdf('seu_arquivo.h5', 'nome_do_grupo')
pd.read_html()
    Descrição: Lê tabelas de uma página da web em um DataFrame do pandas.

    Code
    import pandas as pd
    url = 'https://exemplo.com/tabela.html'
    df_list = pd.read_html(url)
    df = df_list[0] # Assume que a tabela desejada é a primeira na página
pd.read_json()
    Descrição: Lê um arquivo JSON (JavaScript Object Notation) em um DataFrame do pandas.

    Code
    import pandas as pd
    df = pd.read_json('seu_arquivo.json')
pd.read_parquet()
    Descrição: Lê um arquivo Parquet em um DataFrame do pandas.

    Code
    import pandas as pd
    df = pd.read_parquet('seu_arquivo.parquet')

Utilizado a função Loc e Sorted

Função loc:

    A função loc é uma função do pandas utilizada para selecionar e acessar um grupo de linhas e colunas por rótulos ou uma condição booleana. É especialmente útil para a indexação baseada em rótulos.

    import pandas as pd
    dados.loc[dados['Nationality'] == 'Brazil']

    import pandas as pd
    dados.loc[dados['Age'] == 21]
Função sorted:
    A função sorted em Python é utilizada para ordenar iteráveis (listas, tuplas, etc.) em ordem ascendente ou descendente.

    dados.sort_values('Age')
    Vai colocar do menor para o maior


    dados.sort_values('Age', ascending = False)
    Vai colocar do maior para o menor
Ambas as funções são ferramentas poderosas em Python, sendo a 'loc' útil para seleção de dados em DataFrames usando rótulos e condições, enquanto a 'sorted' é fundamental para ordenar iteráveis. Lembre-se de que o uso específico dessas funções dependerá do contexto do seu código e dos seus requisitos.

Exportando Dados:

Quando for exportar dados com Python pode-se gerar em varios formatos sendo:

    CSV (Comma Separated Values):
      dados.to_csv('file.csv', index=False)

    Excel:
      df.to_excel('file.xlsx', index=False)

    JSON (JavaScript Object Notation):
      df.to_json('file.json', orient='records')

    HTML:
      df.to_html('file.html', index=False)

    Parquet:
      df.to_parquet('file.parquet', index=False)

    HDF5 (Hierarchical Data Format version 5):
      df.to_hdf('file.h5', key='data', mode='w')

    Feather:
      df.to_feather('file.feather')

    Msgpack:
      df.to_msgpack('file.msg', compress='zlib')

    Stata:
      df.to_stata('file.dta', version=118)

    SQL (Structured Query Language):
      from sqlalchemy import create_engine
      engine = create_engine('sqlite:///file.db')
      df.to_sql('table_name', con=engine, index=False)
